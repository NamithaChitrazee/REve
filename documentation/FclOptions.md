# Mu2e Event Display Configuration Options (FCL)

This document explains the common configuration parameters for the `Mu2eEventDisplay` module within a FCL (Fermilab Configuration Language) file, used to control what geometry and data are visualized by the offline event display tool.

The configuration is broadly divided into top-level module parameters (which primarily control *what type* of data or geometry to show) and nested `filler` parameters (which control *which specific collections* from the event record are loaded).
